Recipe Buddy - Android App

Recipe Buddy

Recipe Buddy enables users to create, view, and manage their recipes with ease. The app supports image capture via camera, stores recipe data in Firebase Firestore, and retrieves images from Firebase Storage. Google Authentication ensures secure login.

Features:
  • Google Authentication with persistent sessions.
  • Dynamic Recipe Grid with responsive design.
  • Detailed recipe view with a curved image header.
Drawbacks & Improvements:
  • Slow image loading on weak networks.
  • Auto sign-in with last-used account unless logged out manually.
  • Requires active internet connection; consider offline caching.

Role: Developer and Project Manager – responsible for key screen development, project management, and ensuring smooth user navigation.
